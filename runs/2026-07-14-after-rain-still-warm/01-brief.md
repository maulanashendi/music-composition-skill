# Level 1 — Konsep Artistik

## Judul sementara
After Rain, Still Warm

## Brief dan asumsi default
Tidak ada brief musikal dari user setelah permintaan klarifikasi; asumsi ini dicatat eksplisit agar run dapat direvisi, bukan diperlakukan sebagai preferensi user: neo-soul/chill-jazz instrumental original, D minor, 74 BPM, 4/4, 16 bar, tanpa solo panjang.

| Field | Keputusan |
|---|---|
| Gaya | Neo-soul / chill-jazz instrumental |
| Mood | Hangat, lega, sedikit melankolis setelah hujan malam |
| Tempo | 74 BPM |
| Birama | 4/4 |
| Karakter | Laid-back 16th, pocket `neo-soul-core`, ruang antarfrase |
| Instrumentasi | Alto sax, Rhodes, electric bass, drum kit |
| Kompleksitas | Menengah: satu motif, warna modal/minor, satu dominant altered terukur |
| Arah energi | Hening → hangat bergerak → lift terkendali → landing tenang |

## Intent
Satu motif naik minor-third lalu turun langkah membayangkan lampu jalan yang kembali memantul di aspal basah; energi naik lewat tekstur dan register, bukan lewat klimaks keras.
