# Rhythm fundamentals — compact working vocabulary

sebelum menulis pola groove/bass/drum konkret di `groove-rhythm/SKILL.md`.

## Rhythm and subdivision

- **Feel** is set by subdivision and where accents land. Straight-eighths,
  swung-eighths (long-short, ~2:1 loosened toward 3:2 for lofi), and
  16th-based (funk, neo-soul) are the common families.
- **Swing** is a ratio, not a note value — encode it as a parameter, don't
  try to force triplet spelling everywhere.
- **Backbeat** (snare on 2 and 4) anchors most jazz-derived grooves; where
  you *break* it is where you create motion.
- **Pocket** = the consistent micro-timing relationship between layers.
  Bass slightly behind, hats slightly ahead, etc. Decide this at idea stage
  because it defines the groove's identity.
