# Level 14 — Detail Low Level

LEVEL 14 — DETAIL LOW LEVEL
Pada tahap ini, komposisi sudah utuh secara makro. Baru kemudian turun ke detail per bar dan per beat.
Detail melodi
Periksa:

* nada target
* interval
* phrasing
* articulation
* register
* rhythmic placement
* peak note
* repetition
* variation
Detail harmony
Periksa:

* chord spelling
* extension
* alteration
* bass note
* voice leading
* avoid note
* chord-scale implication
Detail rhythm
Periksa:

* syncopation
* rest
* anticipation
* accent
* subdivision
* swing placement
* displacement
Detail voicing
Periksa:

* top note
* inner voice
* spacing
* register
* doubling
* dissonance
* resolution
Detail orchestration
Periksa:

* siapa bermain
* siapa diam
* siapa memimpin
* siapa merespons
* bagaimana transisi terjadi
